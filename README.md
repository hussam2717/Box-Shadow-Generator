
# Box-Shadow-Generator
A simple and interactive **Box Shadow Generator** built with HTML, CSS, and JavaScript.   This tool helps you create beautiful CSS box-shadow effects in real time and copy the generated code instantly.

Live Demo - 

## 🚀 Features
- 🎛️ Live shadow preview
- 🎚️ Control X & Y offset
- 🌫️ Blur & spread adjustment
- 🎨 Color picker for shadow
- 🔁 Real-time CSS output
- 📋 One-click copy CSS code
- 📱 Responsive design (mobile friendly)

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript


## 🎯 How It Works
1. Adjust sliders (X, Y, blur, spread)
2. Choose shadow color
3. See live preview update instantly
4. Copy generated CSS code
5. Paste it into your project
